# 💆‍♀️ Лера про Тело — Система управления подарочными сертификатами

Веб-приложение для учёта и проверки подарочных сертификатов спа-салона **Лера про Тело**.

## ✨ Возможности

- 🔍 **Проверка сертификатов** — введите код или отсканируйте QR
- 📋 **База данных** — просмотр, поиск и фильтрация всех сертификатов
- 🎁 **Создание сертификатов** — генерируйте PDF с QR-кодами
- ✅ **Погашение** — отметьте сертификат как использованный
- 📊 **Статистика** — количество активных, погашенных, общая сумма
- 💾 **Экспорт/Импорт** — резервные копии в JSON
- 📱 **Оффлайн** — работает без интернета (все данные в localStorage)
- 🎨 **Кастомизация** — загружайте собственный фон для сертификатов

## 🚀 Быстрый старт

Приложение доступно по адресу:
```
https://big-art.github.io/lera-certificates/
```

### Локально

1. Клонируйте репозиторий:
```bash
git clone https://github.com/BIG-ART/lera-certificates.git
cd lera-certificates
```

2. Откройте `index.html` в браузере:
```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

Или используйте простой HTTP-сервер:
```bash
python3 -m http.server 8000
# Затем откройте http://localhost:8000
```

## 📖 Использование

### Проверка сертификата
1. Откройте вкладку **"Проверка"**
2. Введите код вручную или нажмите **"Сканировать QR-код"**
3. Приложение покажет статус (активен/погашен)

### Создание PDF-сертификата
1. Откройте вкладку **"Создать"**
2. Введите номинал (или выберите из предложенных)
3. (Опционально) загрузите фон/логотип
4. Нажмите **"Создать PDF-сертификат"**
5. Сертификат автоматически добавится в базу

### Управление
- **Добавить вручную** — создать сертификат без PDF
- **Экспорт** — скачать резервную копию базы (JSON)
- **Импорт** — восстановить базу из файла
- **Сбросить к примерам** — вернуться к тестовым данным

## 🛠 Технологии

- **HTML5** — структура
- **CSS3** — адаптивный дизайн (мобильный-first)
- **Vanilla JavaScript** — логика без зависимостей
- **html5-qrcode** — сканирование QR-кодов
- **jsPDF + html2canvas** — генерация PDF
- **localStorage** — хранение данных

### Изменить цветовую схему

Отредактируйте переменные CSS в начале `<style>`:

```css

    --primary: #d47b5a;        /* основной цвет */
    --primary-dark: #b55e3f;

    --secondary: #a07a62;
    --bg: #fef7f0;             /* фон страницы */
    --text: #3e2a1f;           /* основной текст */

```

```html

    <p>Учёт подарочных сертификатов</p>
</div>
```

## 🔒 Безопасность и приватность

✅ **Все данные хранятся локально в браузере**
- Ничего не отправляется на серверы
- Полная конфиденциальность информации
- Работает полностью оффлайн

## 📊 Структура данных

Сертификаты хранятся в формате:
```json
{
  "code": "W9X3H7K2",
  "amount": "2000 ₽",
  "used": false,
  "usedDate": null,
  "createdDate": "01.06.2025"
}
```

Экспортируйте и импортируйте базу через вкладку **"Управление"**.

## 🐛 Решение проблем

### QR-сканер не работает
- Убедитесь, что используется **HTTPS** (или localhost)
- Разрешите доступ к камере в настройках браузера
- Проверьте версию браузера (Chrome 76+, Safari 13+, Firefox 68+)

### PDF не создаётся
- Попробуйте без фона (сбросьте фон)
- Убедитесь, что введена сумма
- Обновите страницу и повторите попытку

### Данные потеряны после очистки браузера
- Используйте **Экспорт** для создания резервной копии
- Сохраняйте JSON файл в безопасном месте
